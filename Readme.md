# 100 Days CSS Challenge

This repository contains my **100 Days CSS Challenge** journey.

Each day focuses on building small CSS experiments, layouts, and animations to improve my front-end skills.

## Structure

Every challenge is organized in its own folder:

Day_X

- index.html
- style.css
- script.js

## Goal

The goal of this challenge is to:

- Improve CSS layout skills
- Practice animations and creative UI ideas
- Build consistency by coding every day

## Automation

A small shell script (`auto.sh`) is used to automate folder creation for each day.

Example usage:

./auto.sh -f

This creates the folder and boilerplate files for a new challenge day.

## Progress

The repository will gradually be filled with daily experiments and UI components as the challenge progresses.
